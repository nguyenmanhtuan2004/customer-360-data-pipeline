Chapter III · ETL Pipeline
Quy Trình Xử Lý 4 Bước
Step 1A
ETL Log Search
📂 Parquet phân vùng theo ngày
🤖 AI phân loại keyword → thể loại
📏 Window Function tìm keyword phổ biến nhất
✅ Output: 79,732 dòng · 9 cột
Step 1B
ETL Log Content
📂 JSON lồng nhau ~2M dòng/tháng
📊 Pivot theo thể loại AppName
🏷 Gán nhãn Active / Taste / MostWatch
✅ Output: 1,920,545 dòng · 9 cột
Step 2
Data Modeling
🔗 Full Outer Join 2 nguồn → OBT
⭐ Bóc tách 1 Fact + 4 Dim Tables
💾 Lưu Parquet columnar format
✅ Fact: 2,000,277 dòng · 24 cột
Step 3+4
Load + Stored Procedure
☁️ Load dữ liệu lên BigQuery Warehouse
⚙️ SQL Stored Procedure tối ưu hóa
📊 Phân chia phân vùng, cụm dữ liệu
✅ DWH sẵn sàng cho phân tích OLAP
